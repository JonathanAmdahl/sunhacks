# Repo Setup
Monorepo.
Frontend - NextJS, Typescript w/ ESLint, Tailwind
Backend - Typescript w/ Express + MySQL

# Installation Setup
1) cd to backend and frontend respectively
2) npm i
3) setup .env files based on the .env.example
4) run the scripts in package.json
